# Description of the project

- Organize a webinar
  - It should be organized at least 3 days before
  - The maximum number of participant is 1000 and the minimum is 1

# Installation

```bash
$ pnpm install
```

# Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

# Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
